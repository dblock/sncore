Welcome to your new Facebook.NET Application. Follow the steps below to use this project to build your own Facebook application.

PLEASE READ 

NOTE: These steps assume you already have a Facebook account, if not you should create one now. Many of the links below will not work unless you have an FB account. This application includes the Facebook.NET assembly though you should check codeplex for the latest version.

Setting up your application on Facebook using an IFrame*
  1. Add the Developer application to your account (you need an account for that link to work)
  2. Click Developer application from the left column and select click Set Up New Application
  3. Enter the Application Name and expand Optional Fields
  4. Under Base Options
    • Callback URL -> http://localhost:nnnn/MyApp/
      Using localhost makes it easy for you to debug locally
    • Select "Use iframe"
    • Can your application be added to Facebook? -> Yes (this will display additional options)
    • TOS URL -> http://localhost:nnnn/MyApp/Install/TOS.aspx
  5. Under Installation Options
    • Check "Developer Mode"
    • Post-Add URL -> http://localhost:nnnn/MyApp/Install/PostAdd.aspx
    • Fill in your application description
    • Post-Remove URL -> http://localhost:nnnn/MyApp/Install/PostRemove.aspx
  6. Under Installation Options
    • Check "Developer Mode"
    • Post-Add URL -> http://localhost:nnnn/MyApp/Install/PostAdd.aspx
    • Fill in your application description
    • Post-Remove URL -> http://localhost:nnnn/MyApp/Install/PostRemove.aspx
  7. Under Integration Points
    • Site Nav -> http://apps.facebook.com/{appname} basically, use the same URL from Canvas Page URL (see Optional Fields)
    • Privacy URL -> http://localhost:nnnn/MyApp/Integ/Privacy.aspx
    • Help URL -> http://localhost:nnnn/MyApp/Integ/Help.aspx
Accessing your application on Facebook
Once you've created your application you'll want to add it to your FB sidebar so you can use it easily. To do that follow these steps:
  1. Click Save then from the Developers Application browse to My Applications
  2. Click View About Page then click Add Application

Configuring your application

The Facebook Developer platform requires both an APPKEY and a SECRET key that are used for authentication. You need to follow these steps to set up the property variables.

  1. Open both web.config and in the <facebook> section fill in [AppKey] and [Secret] attributes with the values for your application from Facebook.
  2. Fire up the browser and hit your http://apps.facebook.com/{appname}; and you're off!

*What about FBML?

You can use FBML and the Facebook.NET download from CodePlex has an example.
