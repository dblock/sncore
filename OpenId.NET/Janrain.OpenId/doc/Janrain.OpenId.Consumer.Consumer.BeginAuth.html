Consumer.BeginAuth Method 

This method is called to start the OpenId login process.

public AuthRequest BeginAuth(
   Uri userUrl
);

Parameters

userUrl
This is the url the user entered as their OpenId.

Return Value

This method returns an instance of AuthRequest representing the state of the authorization request.

Remarks

To create the userUrl parameter from the text submitted by the user, use the NormalizeUri method of UriUtil.
